Welcome to the Canvac Web store

This application is used to order various product offered, by Canvac. For now, Peptide and Tetramer (along with some special request) can be ordered through the use of this application. Once logged in, the user can also change their account information, and view their order history.

This application is using only "open source" tools and framework such as Tomcat, Struts, Spring, iBATIS
